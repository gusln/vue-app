# huilianyi-weichat

> 汇联易微信项目

git clone  http://hpm.hand-china.com/diffusion/HUILIANYIWECHAT/huilianyi-wechat.git -b develop

## 环境配置

``` bash
# 安装依赖包
npm install

# 调试环境
npm run dev 监听端口8090

# 生成测试环境目标文件
npm run build-test

# 生成生产环境目标文件
npm run build-prod

# 单元测试环境
npm run unit

# run e2e tests
npm run e2e

# run eslint
npm run lint

# run all tests
npm test
```
